# ⚖️ ScreenDiff: Visual Regression

![Screen Diff](../../assets/screenshots/screen_diff.png)

## 🇺🇸 English
### What is it?
ScreenDiff is a specialized tool for kernels with a UI. It compares the visual output of your kernel against a reference image to ensure your graphics drivers are rendering correctly.

### How to use it?
1. Capture a reference screenshot of your kernel's UI when it's perfect.
2. Run your current build.
3. ScreenDiff will overlay both images and highlight any pixel mismatch in bright magenta.
4. Perfect for verifying font rendering, UI alignment, and color accuracy.

---

## 🇪🇸 Español
### ¿Qué es?
ScreenDiff es una herramienta especializada para kernels con interfaz gráfica. Compara la salida visual de tu kernel con una imagen de referencia para asegurar que tus drivers gráficos están renderizando correctamente.

### ¿Cómo usarlo?
1. Captura una captura de referencia de la UI de tu kernel cuando esté perfecta.
2. Ejecuta tu versión actual.
3. ScreenDiff superpondrá ambas imágenes y resaltará cualquier discrepancia de píxeles en magenta brillante.
4. Perfecto para verificar el renderizado de fuentes, la alineación de la interfaz y la precisión del color.
