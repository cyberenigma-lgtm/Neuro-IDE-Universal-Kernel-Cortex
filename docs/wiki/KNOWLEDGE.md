# 🧩 Knowledge Graph: Concept Mapping

![Knowledge](../../assets/screenshots/knowledge_graph.png)

## 🇺🇸 English
### What is it?
The Knowledge Graph turns your codebase into a visual web. It maps the dependencies and interactions between different kernel modules (scheduler, memory manager, disk drivers).

### How to use it?
1. Load your project folder into the IDE.
2. The Graph will automatically identify your source files and their includes.
3. Drag nodes to organize your mental map of the project.
4. Useful for identifying "Single points of failure" where too many modules depend on one file.

---

## 🇪🇸 Español
### ¿Qué es?
El Knowledge Graph convierte tu base de código en una red visual. Mapea las dependencias e interacciones entre los diferentes módulos del kernel (planificador, gestor de memoria, drivers de disco).

### ¿Cómo usarlo?
1. Carga la carpeta de tu proyecto en el IDE.
2. El Gráfico identificará automáticamente tus archivos fuente y sus inclusiones.
3. Arrastra los nodos para organizar tu mapa mental del proyecto.
4. Útil para identificar "puntos únicos de fallo" donde demasiados módulos dependen de un solo archivo.
