# 📡 Neuro-Scope & Divergence Analyzer

![Divergence](../../assets/screenshots/divergence.png)

## 🇺🇸 English
### What is it?
Neuro-Scope is the primary sensory input of the IDE. It captures serial logs at high speed. The **Instruction Divergence** plugin specifically analyzes execution traces to find where two similar runs start to behave differently.

### How to use it?
1. Open the **Neuro-Scope** tab to see your kernel logs.
2. In the **Divergence** sub-tab, load a "Golden Trace" (a known good run).
3. Start your current run.
4. The IDE will highlight exactly which instruction or log message caused the first deviation.

## 🇪🇸 Español
### ¿Qué es?
Neuro-Scope es la entrada sensorial principal del IDE. Captura logs serie a alta velocidad. El plugin **Instruction Divergence** analiza específicamente las trazas de ejecución para encontrar dónde dos ejecuciones similares empiezan a comportarse de forma distinta.

### ¿Cómo usarlo?
1. Abre la pestaña **Neuro-Scope** para ver los logs de tu kernel.
2. En la sub-pestaña **Divergence**, carga una "Traza Dorada" (una ejecución que sepas que funciona).
3. Inicia tu ejecución actual.
4. El IDE resaltará exactamente qué instrucción o mensaje de log causó la primera desviación.
